# Squad Up

A full-stack application for organizing recreational activities and connecting people through shared interests.

## Overview

Squad Up is a mobile application designed to help people discover partners for recreational activities and organize small group events based on shared interests, availability, and safety-focused communication preferences.

Users can create personal profiles, browse local activities, create their own activity postings, join existing events, and communicate through direct messaging and activity-based group chats. The platform is designed to minimize the friction involved in organizing casual recreational activities while keeping the experience simple, approachable, and focused on real-world interaction.

## The Problem

Many people struggle to find activity partners despite having access to community centers, gyms, sports facilities, outdoor recreation areas, and large populations of individuals with similar interests.

Existing solutions often emphasize large organized events, complex group management, or social media features rather than helping users quickly connect and coordinate local activities.

Common pain points include:

- Limited support for casual or spontaneous activity coordination
- Poor local discovery tools
- Complicated event and group management
- Little visibility into who is attending an activity
- Social features that distract from the primary goal of meeting people
- Limited safety controls and communication preferences

Squad Up aims to reduce the friction between discovering activities and meeting people by providing a streamlined platform focused on organizing real-world recreational experiences.

## Core Features

The project is currently being developed around the following core functionality:

- User authentication and profile management
- Activity creation and discovery
- Activity participation and RSVP management
- Direct messaging between users
- Activity-based group chats
- User moderation, blocking, and reporting
- Privacy and communication preference controls
- Shared backend supporting both mobile and web clients

## Current Status

Squad Up is currently under active development.

Current development efforts are focused on building the application's core architecture, including the PostgreSQL database schema, Ruby on Rails backend, REST API, authentication, activity management, messaging infrastructure, and React Native mobile client.

A Vue 3 web client is also being developed to share the same backend and business logic as the mobile application.
